D-CreEA - A Tool to Aid The Process of Creating an Educational Card Game

This tool was developed in a master's thesis in partnership with the Federal University of Ceará

Author: Rubens Silva

What is D-CreEA Card Games?

D-CreEA is software to assist professors in creating educational card games

The games created by the tool can be printed together with the game manual

The professor can customize his game according to its pedagogical content

The big difference of D-CreEA is: the professor does not need to have knowledge in game design

UNLEASH YOUR CREATIVITY

The cards and game decks are customizable: you can choose the fields that will appear on the cards, their scores, the colors

Create the game rules as you wish! The game rules are fully editable

With D-CreEA, the entire process of card prototyping, adjustments, GDD, game loop, etc. are previously set, facilitating the game development

For Non-Experts

To make life easier for professors who have little or no knowledge of game design, we created the following artifacts:

  • A pre-implemented sample game is available to make D-CreEA easier to understand and manipulate
  • Standard Game Design Document (GDD) is provided, you only need to edit
  • Game Loop with drag-n-drop mechanism to aid the edition of game states
  • Card and deck template with predefined fields

Also, you can consult the Guide of Game Creation and the Tutorials available.

Features

  • Game Design Document (GDD)
  • Creation and Customization of Cards
  • Creation and Customization of Decks
  • Game Loop
  • Custom Rules
  • Practical example available
  • Automated Game Manual
  • Cards and Game Manual printing just by a click
